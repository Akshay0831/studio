# Artifex Studio

Artifex Studio - A unified AI-powered creative studio combining real-time image and audio editing capabilities with direct AI model integration.

## Features

- **AI Image Generation**: Real-time sprite and art generation with various AI models
- **Audio Production**: Multi-track audio composition and editing tools
- **Direct AI Integration**: Seamless connection to multiple AI generation services and APIs
- **Real-Time Editing**: Canvas-based manipulation for both visual and audio content
- **Cross-Platform**: Native support for Windows, macOS, and Linux

## 📦 Installation

### Quick Start
1. **Download the latest release** from [GitHub](https://github.com/Akshay0831/Artifex-Studio)
2. **Extract to your desired location**
3. **Run the installer**:
   - **Windows**: Double-click `installer/INSTALL_AND_LAUNCH.bat`
   - **macOS/Linux**: Run `installer/INSTALL_AND_LAUNCH.sh`
   - **Manual**: Run `python3 installer/gui_launcher.py`

### Requirements
- Python 3.8+ with pip

## 🎯 Getting Started

1. **Launch the Application**: Run the installer and follow the prompts
2. **Configure AI Models**: Set up your preferred AI generation services
3. **Start Creating**: Use the AI-powered tools for content generation
4. **Export Results**: Save and export your generated content
